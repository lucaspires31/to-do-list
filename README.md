# 📝 Lista de Tarefas

Esse projeto é uma aplicação simples de lista de tarefas (to-do list) feita em React.

## 💡 Sobre o projeto

A ideia do projeto é permitir que o usuário:
- adicionar tarefas
- visualizar as tarefas na tela
- marcar tarefas como concluídas

Foi desenvolvido para praticar conceitos básicos do React.

## 🧠 Conceitos utilizados

Durante o desenvolvimento eu utilizei:

- Componentes → separando em partes (App, entrada e lista)
- Props → para passar dados e funções entre os componentes
- State → para armazenar as tarefas e controlar o input

## ⚙️ Como funciona

O usuário digita uma tarefa no campo de texto e clica em "Adicionar".

A tarefa aparece na lista com uma caixinha ao lado, onde é possível marcar como concluída.

## 🚀 Tecnologias usadas

- React
- JavaScript
- CSS Module

## 📁 Estrutura básica

- App → componente principal
- EntradaTarefa → onde o usuário digita
- ListaTarefas → onde as tarefas aparecem
